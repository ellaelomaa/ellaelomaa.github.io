NLP

Project Overview

Helmin NLP-ohjelma is a natural language processing tool for Finnish texts. The program calculates some basic statistics, such as sentence length by graphemes and words, most used words, and unique words. The program also calculates information about morphemes, uses lemmatization to analyze lexical density, and a employs few different algorithms to compare the corpora writers. The results are saved in a .xlsx-file.

In this project the user supplies the corpora in text files, one folder per each writer. Previously I've done a similar program, where the corpora came from Project Gutenberg, fetched with BeautifulSoup.

Main view

Main view of the app. From the left tool bar item the user can either read information about the application or quit the program, and from the right item the user can change the colour theme. On the main window, on the left are all the different natural language processing functions for the user to choose from. On the right side are the instructions and a list of missing/future features.

In progress

Main view when in progress; the colour theme also has been changed. When the program is in progress, the start button is disabled. The download bar length is calculated from the corpora size and the program functions chosen by the user. The GUI also informs the user how many tasks have been done, how many there are in total, and what phase the program is currently busy doing.

Error message

As the application will crash when trying to edit an already opened file, the program checks that the resulting .xlsx- file is closed before starting the program. In case the user gets naughty or bored, and tries to open the file when the program is running, the application also checks the file before saving. In both cases the same error dialogue is shown, and the application will not continue before closing the file.
