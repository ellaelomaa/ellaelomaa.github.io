Notes application

Project Overview

Menagerie is a notes taking, or list making, application made for my own purposes and for practice. The idea came from my love for tea and the local teashop, where I couldn't always remember what varietes I'd already tried and what my opinion was of them - so I started working on an application to remedy that.

The application is being made in Flutter, which I was already familiar with. The database is implemented with SQFlite, a plugin for the popular and lightweight SQLite database. State management is handled with Provider. All images are from public domain sources.

This project not only demonstrates my understanding of relational dabases and state management, but also why I am not majoring in graphic design or UX.

Gallery

Tarot deck

Tarot deck view. Cards are added by sliding it to the left ("past"), down ("present"), or right ("future"). Cards can be deleted and edited.

Adding and editing a note

Notes are added to folders, with one default folder ("Miscellaneous") already created. The same form is used when adding and editing a note, with read-only mode toggled by the eye icon on the top right corner.

Notes list

Notes are sorted by category, and the user can further sort the notes by when they were added or modified, or alphabetically. All orders can be ascending or descending. Furthermore, notes can be pinned, so that they always stay on top of the list.
