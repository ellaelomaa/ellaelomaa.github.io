Kitsain

Project Overview

The application was made as part of the spring 2023 project course at Tampere University. Kitsain is a Helsinki based start-up venture aiming to release an application that allows the user to track their pantry items' expiration dates, nutritional qualities, and cost. Additional features are shopping lists, price comparisons, social media features, with many more planned.

Our project was MVP number two. Of the aforementioned features we implemented the pantry item tracking, shopping lists, branding, GUI, and the history tab, which allows the users see what items were either used or discarded each month. I was in charge of designing, implementing, and testing all database related features in the application. The database I chose was Realm, which is a mobile-first object database management system. The database can also be synced with Mongo Atlas, which could be useful in future development. While the final MVP did not include nutritional information, data was retrieved from Open Food Facts and included in the database model.

Gallery

Item card

Includes item name, category, date opened, expiration date, button to mark/toggle favorite item, a notes box, and a "more" button to either edit the item, delete the item, or mark it as either used or wasted.

Card colours

The colour bar on the left indicates the expiration status: black for expired items, red for items expiring today, yellow for items expiring within the week (also included the date), green for items expiring in more than a week, and grey for items without an expiration date.

Delete item

Deleting an item has a warning message, as the item is deleted from the database and cannot be retrieved again.

Categories

Pantry view when sorted by category.

Filtering options

Besides showing all items by category, filter options include all items, opened itemes only, or items marked as favorite.

Sorting options

Regardless of how the items are filtered, the items can be sorted alphabetically, by nearest expiration date, or by the order they were added in.

Pantry history

In the pantry view, items can be marked as either used or wasted. In the history page there are tabs for both. For each month, the application counts how many of the history items were in either category. In this picture, in January 2023 two items were moved to history tab, of which 50 % was marked as used - in this case a banana.
